# incidentes/

Incidentes reportados (PROD u otros entornos) y diagnosticos relacionados.

## Estados

- [abiertos/](abiertos/) — incidentes con accion pendiente.
- [diagnosticos/](diagnosticos/) — analisis tecnicos no concluidos (puede o no haber incidente formal).
- [cerrados/](cerrados/) — incidentes resueltos. Mover aqui solo cuando esten oficialmente cerrados.
