# Menu APEX - Agrupar "Certificado Digital" y "Onboarding" bajo "Productos Digitales"

## Descripcion

Yo como negocio, requiero que las opciones de `Certificado Digital` y `Onboarding` se agrupen dentro de un mismo submenu llamado `Productos Digitales`, para simplificar el menu lateral y reflejar que ambas pertenecen al mismo modulo de productos digitales del cliente.

## Criterios de aceptacion

- El menu lateral debe incluir una nueva entrada llamada **"Productos Digitales"**.
- Las opciones **"Certificado Digital"** y **"Onboarding"** deben aparecer dentro del submenu **"Productos Digitales"** y dejar de mostrarse como entradas de primer nivel.
- El submenu **"Productos Digitales"** debe mostrarse con un icono representativo de productos.
- Al hacer click en cada opcion del submenu, el sistema debe abrir la misma pantalla a la que se accedia antes del cambio.
- El acceso a cada opcion debe seguir respetando los permisos actuales de cada usuario.
