# _promociones/

Registro cronologico de eventos de promocion entre entornos. Cada evento es un archivo `YYYY-MM-DD_<descripcion>.md` con:

- Origen y destino.
- Objetos incluidos.
- OPT / historia asociada.
- Evidencia previa al pase.
- Resultado posterior al pase.

Estos eventos son auditables y no se mueven aunque la carpeta de la historia cambie de estado.
