# Instrucciones del agente

- **Ámbito principal:** Todas las lecturas y cambios deben realizarse dentro de `ADEMI_BD/ENTORNOS_ORACLE/DESARROLLO`.
- **Ubicación de entregables:** Colocar los scripts generados para retorno en `ADEMI_BD/ENTORNOS_ORACLE/DESARROLLO/scripts_retorno`.
- **Herramientas recomendadas:** usar `rg` para búsquedas, `sed` para inspección, y `apply_patch` para ediciones puntuales.
- **Registro de decisiones:** documentar en las respuestas la ruta de cada archivo tocado y el propósito del cambio o script entregado.
- **Validaciones previas:** antes de proponer ejecución en producción, indicar claramente prerequisitos, parámetros y cualquier dependencia externa.
