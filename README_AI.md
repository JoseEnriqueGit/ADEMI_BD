# README_AI

## Uso rápido
- Fuente de verdad compartida: `docs/instrucciones_ai/BASE_OPERATIVA.md`
- Referencias por tipo de objeto: `docs/instrucciones_ai/referencias/`
- Prompts listos para Codex: `docs/prompts_codex/`
- Comandos principales de Claude: `/explicar`, `/optimizar`, `/comparar_entornos`, `/incorporar_objeto`

## Regla más importante
Si no indicas entorno, el asistente debe preguntarte antes de leer objetos Oracle.

## Atajos mentales
- `explicar`: entender un objeto concreto
- `optimizar`: diagnosticar, medir alcance y proponer mejoras sin cambiar lógica
- `comparar_entornos`: contrastar QA y DESARROLLO, incluyendo dependencias inmediatas si es relevante
- `incorporar_objeto`: agregar al repo un objeto faltante en su ruta real
