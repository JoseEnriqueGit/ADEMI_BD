# explicar_objeto

## Cuándo usarlo
Cuando quieras que Codex explique un package, tabla, vista, procedure, job u otro objeto Oracle.

## Prompt base
```text
Analiza y explica el objeto Oracle [NOMBRE_O_RUTA] en el entorno [ENTORNO].

Quiero una respuesta en español, técnica y corta.
Lee el objeto completo antes de responder.
Incluye archivo y líneas exactas.
Explica dependencias, flujo y lógica de negocio.
Si ves diferencias relevantes con otro entorno, avísame.
Si falta alguna dependencia, detente y pídemela.
```

## Ejemplo
```text
Analiza y explica el package PR_PKG_REPRESTAMOS en el entorno QA.

Quiero una respuesta en español, técnica y corta.
Lee el spec y el body completos antes de responder.
Incluye archivo y líneas exactas.
Explica dependencias, flujo y lógica de negocio.
Si ves diferencias relevantes con DESARROLLO, avísame.
Si falta alguna dependencia, detente y pídemela.
```
