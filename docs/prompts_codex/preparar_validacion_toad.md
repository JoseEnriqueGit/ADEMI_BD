# preparar_validacion_toad

## Cuándo usarlo
Cuando quieras que Codex te deje pasos claros para validar un análisis o cambio en Toad.

## Prompt base
```text
Prepara una guía de validación en Toad para [OBJETO_O_CAMBIO].

Quiero pasos concretos y cortos.
Incluye compilación, dependencias, prueba funcional y verificación de rendimiento cuando aplique.
Aclara qué sí se observó en el repo y qué solo se puede validar en la base de datos.
```

## Ejemplo
```text
Prepara una guía de validación en Toad para la optimización del job de cancelado en PR_PKG_REPRESTAMOS.

Quiero pasos concretos y cortos.
Incluye compilación, dependencias, prueba funcional y verificación de rendimiento cuando aplique.
Aclara qué sí se observó en el repo y qué solo se puede validar en la base de datos.
```
