# incorporar_objeto

## Cuándo usarlo
Cuando falte un objeto en el repo y quieras agregarlo en su ruta real.

## Prompt base
```text
Voy a compartirte un objeto Oracle faltante para incorporarlo al repositorio.

Ubícalo en la ruta real según entorno, schema y tipo de objeto.
No uses carpetas temporales si ya se conoce el destino real.
Déjalo normalizado de forma completa, sin cambiar su lógica.
Incluye una cabecera con entorno, schema, fecha, origen, motivo/caso y observación breve.
Si falta decidir la trazabilidad adicional en historias, pregúntame.
```

## Ejemplo
```text
Voy a compartirte una tabla faltante de QA para incorporarla al repositorio.

Ubícala en la ruta real según entorno, schema y tipo de objeto.
No uses carpetas temporales si ya se conoce el destino real.
Déjala normalizada de forma completa, sin cambiar su lógica.
Incluye una cabecera con entorno, schema, fecha, origen, motivo/caso y observación breve.
Si falta decidir la trazabilidad adicional en historias, pregúntame.
```
