# bitacora

## Cuándo usarlo
Al cerrar un trabajo relevante, para dejar memoria cronológica en el repo.

## Prompt base
```text
Agrega una entrada a docs/memoria/BITACORA.md.

1. Usa la plantilla docs/memoria/_plantillas/ENTRADA_BITACORA.md.
2. Agrega un bloque corto al inicio del archivo (más reciente arriba), sin editar entradas pasadas.
3. Si cambió el panorama abierto/vivo, actualiza docs/memoria/CONTEXTO_ACTUAL.md.
4. Si cambió el estado de un caso, actualiza historias/INVENTARIO.md y el ESTADO.md del caso.

Registra: objetivo, qué se hizo, decisiones, pendientes y archivos/commits tocados.
```
