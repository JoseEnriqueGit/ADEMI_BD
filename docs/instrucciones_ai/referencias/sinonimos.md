# sinonimos

## Qué leer
- definición completa del sinónimo
- objeto objetivo si existe en el repo

## Dependencias a buscar
- schema real del objeto
- consumidores que usan el nombre lógico

## Qué validar
- qué objeto abstrae
- si puede ocultar dependencias externas
- diferencias entre entornos

## Plantilla de respuesta
1. Qué referencia
2. Para qué sirve
3. Quién lo usa
4. Riesgos o ambigüedades
5. Validación sugerida

## Reglas de optimización
- no optimizar el sinónimo aislado; optimizar el objeto real o sus consumidores
- usarlo para aclarar dependencias, no para asumirlas

## Tips de Toad
- revisar el objeto apuntado y sus consumidores
