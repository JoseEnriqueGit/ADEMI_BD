# bitacora

Agrega una entrada a la memoria cronologica del proyecto.

Pasos:
1. Lee la plantilla `docs/memoria/_plantillas/ENTRADA_BITACORA.md`.
2. Agrega un bloque corto al **inicio** de `docs/memoria/BITACORA.md` (mas reciente arriba),
   sin editar ni borrar entradas pasadas.
3. Si cambio el panorama abierto/vivo, actualiza `docs/memoria/CONTEXTO_ACTUAL.md`.
4. Aplica las reglas de la skill `memoria-engram`.
